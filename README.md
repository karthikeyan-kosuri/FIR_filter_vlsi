### VLSI Design of FIR Filter for DSP Applications

This project implements a Finite Impulse Response (FIR) filter in Verilog for digital signal processing applications. Filter coefficients are generated using MATLAB and integrated into the hardware design, which is then simulated and synthesized to evaluate performance metrics such as area, speed, and power consumption.

* Generated FIR filter coefficients using MATLAB.
* Implemented the FIR filter architecture in Verilog using shift registers, multipliers, and adders.
* Simulated and synthesized the design using FPGA/ASIC design tools.
* Analyzed and compared area, timing, and power utilization results.
